# Polymarket BTC 5-Minute Signal Intelligence Platform

## Project Overview

**Project Name**: PolySignal BTC - Polymarket BTC 5-Minute Market Intelligence Platform  
**Project Type**: Premium Flask SaaS Web Application  
**Core Purpose**: Real-time signal generation, market tracking, and advanced analytics for Polymarket's BTC 5-minute "Up or Down" binary prediction markets  
**Target Users**: Traders and analysts interested in short-term BTC direction prediction market intelligence

- **Database**: SQLite with SQLAlchemy ORM
- **API**: REST API with JSON responses

## Core Features

### 1. Market Detection & Tracking
- Detect active Polymarket BTC 5-minute market via Gamma API
- Parse market title to extract window start/end times
- Retrieve opening "Price to Beat" reference price
- Fetch live implied Up/Down probabilities
- Track market status (upcoming, live, resolved)
- Store market snapshots every refresh cycle

**Data Model**:
```python
class MarketSnapshot:
    - market_id: str (Polymarket condition ID)
    - event_title: str
    - window_start: datetime
    - window_end: datetime
    - price_to_beat: float (opening reference price)
    - live_price: float (current BTC price)
    - up_probability: float (0-1)
    - down_probability: float (0-1)
    - status: str (upcoming/live/resolving/resolved)
    - snapshot_time: datetime
    - volume_usd: float
```

### 2. Signal Generation Engine
Generate signals with weighted confluence model:

**Signal Output**:
```python
Signal:
    - direction: UP | DOWN | NO_TRADE
    - confidence: 0-100
    - tier: weak(0) | moderate(1) | strong(2) | elite(3)
    - reasoning: list of factor explanations
    - timestamp: datetime
    - market_id: str
    - factors: dict of contributing signals
```

**Signal Factors (Weighted)**:
1. **Polymarket-specific (40%)**
   - Price distance from Price to Beat (+15%)
   - Time remaining decay (+5%)
   - Polymarket implied probability skew (+10%)
   - Odds movement momentum (+5%)
   - Orderbook imbalance (+5%)

2. **BTC Microstructure (35%)**
   - 1-minute candle momentum (+8%)
   - EMA 9/20/50 alignment (+7%)
   - RSI level (+5%)
   - MACD histogram (+5%)
   - ATR volatility context (+5%)
   - Bollinger Band position (+5%)

3. **Risk Filters (25% negative weight)**
   - Chop index detection (-10%)
   - Conflicting signals penalty (-5%)
   - Thin liquidity warning (-5%)
   - Late entry penalty (-5%)

**Confidence Tiers**:
- elite: 90-100
- strong: 80-89
- moderate: 70-79
- weak: 50-69
- NO_TRADE: <50

### 3. Signal Resolution Tracking
- Automatically evaluate signals when markets resolve
- Fetch actual outcome from Polymarket
- Compare prediction vs outcome
- Calculate correctness and edge

**Data Model**:
```python
class SignalLog:
    - signal_id: UUID
    - market_id: str
    - timestamp: datetime
    - signal_direction: UP | DOWN | NO_TRADE
    - confidence: int
    - price_to_beat: float
    - live_price_at_signal: float
    - reasoning: str
    - regime: str (trending/sideways/breakout/whipsaw)
    - resolved_outcome: UP | DOWN | null
    - is_correct: bool | null
    - created_at: datetime
```

### 4. Daily Analytics
**Today Stats**:
- Total signals, UP/DOWN/NO_TRADE counts
- Resolved vs pending
- Correct/Wrong/Skipped counts
- Gross win rate (correct/resolved)
- Actionable win rate (correct/(correct+wrong))
- Average/median confidence
- Current streak, max win/loss streak

**Advanced Analytics**:
- Accuracy by confidence bucket (50-59, 60-69, 70-79, 80-89, 90+)
- Accuracy by hour of day
- Accuracy by regime
- UP vs DOWN performance
- Final minute vs earlier entries
- Rolling 10/20/50 accuracy
- Confusion matrix breakdown

## Known Limitations

1. **Btc Price Source**: Will use Binance as supplementary since Chainlink direct feed requires specific API access. The app will note this distinction clearly.

2. **Historical Data**: Polymarket may not provide historic 5-minute market data. Will implement synthetic backtest using Binance candles if needed.

3. **Future Leakage**: Strictly avoid by only using data available at signal time.

4. **No Trading**: This is an analytics platform, not a trading bot.
